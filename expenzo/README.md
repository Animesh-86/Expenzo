# Expenzo 💰

Expenzo is a premium, high-performance personal finance manager built with Flutter. It focuses on visual excellence, speed, and robust data integrity to help you master your money.

![Expenzo Logo](assets/images/app_icon.png)

## 🚀 Key Features

*   **Savings Goals (NEW)**: Track your progress towards financial milestones with smart weekly target calculations and deadlines.
*   **Intuitive Expense Tracking**: Log expenses instantly with categories and descriptions. Now supports **Editing** of existing entries!
*   **Smart Analytics**: Visualize your spending trends with beautiful charts and category-wise breakdowns.
*   **Budget Management**: Set spending limits to stay on top of your financial health.
*   **Robust Data Stability**: Built with a self-healing Hive database architecture to prevent data corruption.
*   **Dark Mode & Glassmorphism**: A sleek, premium UI designed for modern aesthetics.
